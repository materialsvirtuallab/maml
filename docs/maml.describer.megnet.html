class maml.describer.megnet.MEGNetStructure(name: Union[str, megnet.models.base.GraphModel, None] = None, mode: str = 'site_stats', level: Optional[int] = None, stats: Optional[List] = None, **kwargs)¶

Bases: maml.base._describer.BaseDescriber

Use megnet pre-trained models as featurizer to get structural features. There are two methods to get structural descriptors from megnet models.

mode:
‘site_stats’: Calculate the site features, and then use maml.utils.stats to compute the feature-wise

statistics. This requires the specification of level

‘site_readout’: Use the atomic features at the readout stage ‘final’: Use the concatenated atom, bond and global features
